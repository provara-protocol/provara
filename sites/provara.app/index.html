How It Works

  1. Create Your Vault You run a command: provara init. This generates Ed25519 signing keys and creates a genesis event ‚Äî your vault's birth certificate.
  2. Write Memories You add entries to your vault: family events, decisions, observations. Each entry is timestamped and cryptographically signed.
  3. Verify Integrity Run provara verify. The system checks every signature, every hash, every causal chain. Any tampering is detected immediately.
  4. Backup and Sync Copy your vault anywhere: USB drive, cloud storage, another machine. Provara can merge changes intelligently because truth is recomputed from events, not merged directly.
  5. Read Forever Your vault is JSON. Open it with a text editor in 2024 or 2074. The format will still make sense.

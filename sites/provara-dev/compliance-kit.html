Conformance Kit v1

The Conformance Kit verifies whether a candidate Provara-compatible vault is conformant to v1.0 behavior. Source: docs/CONFORMANCE_KIT.md.

Single-command verification

python tools/conformance/verify_impl.py --vault /path/to/vault

Checks include structure, event chain and signatures, reducer determinism, vectors, and optional 17-test compliance execution.

Useful flags

  • --skip-vectors
  • --skip-compliance
  • --json

Expected verification stages

  1. Vault layout and required files
  2. Event schema and causal chain integrity
  3. Signature and key authority validation
  4. Reducer replay determinism
  5. Vector alignment with test_vectors/vectors.json
  6. Compliance suite (optional but required for full conformance)

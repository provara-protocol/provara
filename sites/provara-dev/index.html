Provara Protocol v1.0

Open specification for append-only cryptographic evidence. Provara combines Ed25519 signatures, SHA-256 hashing, and deterministic replay so independent implementations reach byte-identical state.

RFC 8032 (Ed25519) FIPS 180-4 (SHA-256) RFC 8785 (Canonical JSON) Apache 2.0

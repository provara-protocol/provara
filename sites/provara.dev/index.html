Provara Protocol v1.0

A self-sovereign, cryptographically-signed event log designed for cognitive continuity, tamper-evidence, and 50-year readability. No vendor lock-in. No accounts. No servers. Just verifiable truth.

Read the Spec GitHub Conformance Kit
Golden Rule: Truth is not merged. Evidence is merged. Truth is recomputed.

Every event is cryptographic evidence. When combining vaults from multiple sources, you merge the raw observations, then rerun the deterministic reducer to derive fresh conclusions. Merge conflicts at the belief layer are impossible.

Design Guarantees

No Vendor Lock-In

Everything is plain text: JSON events, Python scripts. No proprietary formats. Read your vault forever with a text editor.

No Internet Required

Works entirely offline after initial setup. No phone-home, no telemetry, no cloud dependency. Your data never leaves your machine.

No Accounts

Your identity lives in your files, not on a server. No signup, no login, no password. Ed25519 keys are your proof of authority.

Tamper-Evident

Merkle trees, Ed25519 signatures, and causal chains detect any modification. A single flipped bit fails verification.

Human-Readable

The event log is NDJSON — open it with any text editor and read it. Hashes and signatures are verifiable by hand if needed.

50-Year Readable

JSON, SHA-256, and Ed25519 are industry standards implemented in every language. They will outlive any platform.

Who Is This For?

Individuals Preserving Family Records

Self-sovereign memory. Your family history is yours alone. No platform dependency, no account cancellation, no terms-of-service changes. Cryptographically verifiable at any time.

AI Agents & Embodied Systems

Cognitive continuity across sessions. An AI agent can maintain tamper-evident memory across multiple embodied systems or training epochs. Deterministic reducer enables reproducible reasoning from raw observations.

Organizations Requiring Audit Trails

Tamper-evident event logs for compliance, governance, and forensics. Every event is signed and causally chained. No record can be deleted or modified after the fact.

AI Governance Frameworks

Policy enforcement and decision logging. Record model evaluations, prompt inputs, routing decisions, and safety checks as immutable evidence. Auditors can replay the full decision chain independently.

Core Concepts

Vault
A self-contained directory containing all identity, events, policies, and manifest. Can be replicated, merged, or verified independently.
Event
An immutable record: who, what, when, signature. Events form a causal chain per actor. Once written, events are never modified or deleted.
Reducer
A pure, deterministic function that processes events to derive beliefs. Same events always produce the same state hash, on any machine, forever.
Manifest
A signed inventory of all files in the vault, plus their SHA-256 hashes. Enables integrity verification and tamper detection.
Merkle Root
The cryptographic anchor of the entire vault. A single hex string that changes if any file is modified. Enables fast integrity checks.
Four Namespaces
canonical/ (attested), local/ (node-local), contested/ (conflicting evidence), archived/ (superseded).

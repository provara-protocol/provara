The Golden Rule

"Truth is not merged. Evidence is merged. Truth is recomputed."

Provara vaults are append-only. Integrity is verified cryptographically, never assumed. State is derived by replaying evidence through a deterministic reducer.

Core Invariants

Chain Integrity

Every event is causally linked via SHA-256 hashes. Breaking one link invalidates the entire subsequent chain.

Non-Repudiation

All events are signed with Ed25519. The signature proves provenance and prevents unauthorized modifications.

Determinism

The reducer is a pure function. Given the same events, any compliant implementation produces the exact same state hash.

Auditability

The complete history of observations and decisions is preserved. Auditors can replay the vault to any point in time.

How It Works

1. Ed25519 Signing: Each event is signed with a cryptographic key. The signature proves it came from this vault.

2. SHA-256 Chain: Each event's hash is chained to the previous one. Modifying any event breaks the chain.

3. Verification: Anyone can verify the chain without special tools‚Äîjust the math.

Result: A permanent, tamper-evident record that outlasts any single system.
